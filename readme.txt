== Description ==
 
It a awosome slider plugin. That use bootstrap 4 origianl coursel feature and make awosome customize with functional.

Use Shortcode [boot_slider id=" "] or full [boot_slider id=" " cat=" " speed="3000" caption="show/hide"]

= Here =

[boot_slider id="__any number/text__" cat="__Boot Slider Category Slug Name__" speed="__slider milli scound speed__" caption="__Default it show, for hide write 'hide'__"]
 
== Features ==  

* Bootstrap 5 Slider
* Responsive Design
* Any Wordpress theme supported
* Full Function on Dashboard
* Easy to Use by Shortcode
 
 
== Installation ==
 
This section describes how to install the plugin and get it working.
 
e.g.
  
1. Upload `boot-slider` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `&lt;?php do_shortcode('[boot_slider id=" " cat=" "]'); ?&gt;` in your templates
